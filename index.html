NumViewer

Mettre n'importe quelle image sur ta calculatrice

Tutoriel

Etape 1

Convertir ton image

Utilise l'outil ci-dessus pour configurer ton image. Elle apparaîtra sur ta calculatrice exactement de la même manière que sur le site. Une fois fini, clique sur "Copier le code". Un programme python sera alors copié dans ton presse-papiers.

Etape 2

Créé le script sur ton compte numworks.

Rends-toi sur le site de numworks. Connectes-toi à ton compte (si tu n'en a pas, tu peux en créer un gratuitement). Continue dans "mes scripts".

Puis, créé un nouveau script.

Donne-lui un nom, peu importe tant qu'il finit par ".py". Il te suffit de coller dans l'éditeur de code ton programme copier précédemment.

→

Sauvegarde ton script.

Attention ! Ne te fit pas a l'émulation du script, la preview ne marche pas toujours.

Etape 3

Envoie et teste ton script !

Connecte ta calculatrice, puis clique sur "Envoyer sur ma calculatrice"

Si c'est la première fois, tu devras installer un driver. Suis simplement les instructions du site.

Etape 4

C'est fini !

À toi d'être créatifs. Si ton programme s'arrête sur une page blanche sans afficher l'image. Essaie de réduire la taille de celle-ci, elle est sans doute trop grande.
